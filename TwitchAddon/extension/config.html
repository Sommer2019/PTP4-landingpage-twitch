🪙 Kanalpunkte Extension – Setup

Konfiguriere die Verbindung zu deinem Backend (EBS).

Voraussetzungen

  1. Der TwitchAddon Java-Bot muss laufen (Port 8081).
  2. Der Bot muss über HTTPS öffentlich erreichbar sein (z.B. via ngrok oder eigenem Server).
  3. Die Twitch Extension muss im Twitch Developer Dashboard registriert und aktiviert sein.

Einrichtungsschritte

  1. Öffne das Twitch Developer Dashboard und erstelle eine neue Extension.
  2. Wähle als Typ: Panel (und optional Mobile).
  3. Trage unter Panel Viewer Path folgendes ein:
    /extension/panel.html
  4. Trage unter Config Path:
    /extension/config.html
  5. Trage unter Mobile Path:
    /extension/mobile.html
  6. Als Testing Base URI trägst du die öffentliche HTTPS-URL deines Java-Backends ein (z.B. https://dein-server.example.com).
  7. Starte die Extension auf deinem Kanal über Manage Overlays & Extensions im Creator Dashboard.
💡 Tipp: Mit ngrok http 8081 kannst du deinen lokalen Bot schnell über HTTPS erreichbar machen – ideal zum Testen.

API-Endpunkte (EBS)

  • GET /api/points?user_id=<twitch_user_id> – Punkte eines Zuschauers
  • GET /api/leaderboard?limit=10 – Top-10 Rangliste
  • GET /api/rewards – Alle verfügbaren Rewards
